Análisis de Rendimiento en League of Legends y Test de Hipótesis
Determinar mediante estadística aplicada si alcanzar un Creep Score (CS) mayor a 200 garantiza un porcentaje de victorias de 70%.
Dataset propio construido a partir del historial de partidas (n=32), transformando variables categóricas a numéricas y estandarizando formatos de tiempo.
