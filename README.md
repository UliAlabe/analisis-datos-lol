# 🎮 Análisis de Datos: League of Legends

Aplicación de consola en Python para analizar mi historial de partidas de LoL. Utiliza `pandas` para la limpieza y análisis estadístico de los datos, y `matplotlib` para generar gráficos. 

El objetivo es cruzar las estadísticas de las partidas y usar datos reales para ver qué métricas (CS, KDA, Kills) impactan más en el winrate.

## 🛠️ Tecnologías
- Python 3
- Pandas (Matemática y manipulación de datos)
- Matplotlib (Visualización)

## 📊 Funcionalidades
El script cuenta con un menú interactivo que permite calcular y visualizar:
1. Promedio de CS/min (Global y agrupado por Rol).
2. Winrate porcentual por Campeón.
3. Gráfico de barras de los campeones más jugados.
4. Gráfico ordenado del promedio de kills por campeón.
5. Top 5 de mejores partidas calculando el KDA.
6. Matriz de correlación para evaluar qué factor influye más en las victorias.
